Financial market as a sparse network

Introduction

In this article we show how current methodologies in Statistical Learning and Economics can be combined to represent the stock market as a sparse graph. In particular, we introduce the reader to the concept of sparse graph and its application to financial data. The article is organized as follow: in the first part we describe the data. In the second part we provide a theoretical overview of the techniques used. In the last part we provide the result from the empirical analysis with replicable code.

Data

The data that have been used are stock data dowloaded from the API service of Yahoo! Finance, in the period 2000-2016. The data collected include the first 1000 companies listed in NASDAQ and NYSE with highest market capitalization. After a data cleaning process the total data have been reduced to around 400 stocks. The data include daily percentage change in adjusted price per each company over more then 4000 observations. Along the analysis, observations with more then 1/3 missing values in the periods which has been analyzed have been dropped. For downloading the data click here.

Missing value

We observed that missing values were often due to the fact that companies enter or exit the market before or after the period which has been analyzed. Missing values have been replaced with random independent numbers generated from a Normal distribution centered on zero and with standard deviation corresponding to the standard deviation of the given stock.

Theoretical framework

Let \(X \in \mathcal{R}^{T \times p}\) be the input matrix with \(p\) companies and \(T\) observations over time and let \(\textbf{x}_t\) be the vector of observations at time \(t\). We consider the following regression model: \[ x_{t,i} = \beta_{0,i} + \beta_{1,i} r_{t,m} + \epsilon_{t,i} \] where \(\textbf{r}_m\) is the vector of market returns and \(\boldsymbol{\epsilon}_{t} \sim \mathcal{N}(0, \Sigma_t)\) being the residual errors with \(\epsilon_{t,i} \perp r_{t,m}\). The equation relates to a single factor model, with the residuals of the stock returns being correlated after adjusting for the market effect. Let \(K = \Sigma^{-1}\) be its precision matrix. As adjency matrix \(W\) we consider the matrix containing the partial correlation effect after adjusting for the market effect. Importantly, it can be show[2] that
\[ W_{i,j} = - \frac{K_{i,j}}{\sqrt{K_{i,i} K_{j,j}}} \] As an introductory and simple analysis we assume that \(\Sigma_{2000:2007} = \Sigma_1\), \(\Sigma_{2007:2009} = \Sigma_2\) and \(\Sigma_{2009:2016} = \Sigma_3\). Therefore, we assume that the covariance matrix is constant over time but it changes only before, during and after the crisis. Whereas we could potentially introduce a more complex behaviour of the covariance matrix over time, we believe that this approach might be helpful for a first semplicistic and non-comprehensive understanding of the market structure.

Inducing sparsity

As a further assumption in this context we impose sparsity conditions. Therefore: \[ ||K||_0 \le s^* \] Where \(s^* < p\) represents the upper bound on the true number of non-zero coefficients. Under this condition we force some of stocks to have zero partial correlation. The assumption of sparsity is crucial in order not to have a fully connected network. To tackle this problem we use Graphical Lasso procedure [3]. Graphical Lasso solves a generalized Maximum Likelihood estimation under the assumption of Gaussianity of the Data Generating Process and it adds the L1 penalty as a relaxation for the L0 penalty. The objective function is: \[ \hat{K} = argmin_{K: \in \mathcal{S}^n_{++} } \quad \text{tr}(\Sigma K) - \text{log}(\text{det}(K)) + \lambda \sum_{i \neq j} |k_{i,j}| \] with \(K\) being a positive definite matrix and where \(\text{tr}(\Sigma K) - \text{log}(\text{det}(K))\) is the Normal log-likelihood expressed in term of \(K\) and \(\lambda \sum_{i \neq j} |k_{i,j}|\) is the L1 penalty. The tuning parameter \(\lambda\) is estimated using Cross Validation.

Analysis

We estimate the network from 2000 to 2007. The shrinkage parameter \(\lambda\) for Graphical Lasso is computed using cross validation. We show that a sparse representation of the network provides interesting insights on the connected components of the network itself.

## cross validation for glasso

glasso_cv <- function(ts, k=5, rholist=NULL, verbose=T) {
  library(glasso)
  library(cvTools)
    
  if (is.null(rholist)) {
    # We will determine the maximum rho as one where the inverse covariance
    # has at least 5% non-zero off-diagonal elements (arbitrary)
    S       <- cov(ts)
    rholist <- seq(0, max(abs(S)), length=11)[-1]
    GLP     <- glassopath(S, rholist, trace=0)
    nonzeros<- apply(GLP$wi, 3, function(x) mean(x[upper.tri(x)]!=0))
    max.rho <- max(rholist[nonzeros > 0.05], rholist[1])
    
    # Now make the list of rhos
    rholist <- seq(0, max.rho, length=100)
  }
  
  n     <- nrow(ts)
  folds <- cvFolds(n, k, type="consecutive")
  
  loglikes <- sapply(1:k, function(ki) {
    if (verbose) cat("Fold ", ki, "\n")
    S_train <- cov(ts[folds$which!=ki,])
    S_test  <- cov(ts[folds$which==ki,])
    GLP     <- glassopath(S_train, rholist, trace=0)
    loglike <- apply(GLP$wi, 3, function(P_train) log_likelihood(P_train, S_test))
    loglike
  })
  
  ind     <- which.max(rowMeans(loglikes))
  rhomax  <- rholist[ind]
  S       <- cov(ts)
  a       <- glasso(S, rhomax)
  a$rhomax <- rhomax
  
  return(a)
}

Conclusion

Clearly, the method represent well the structure of the market. In particular we notice tha cliques and connected components are referred to firms in the same industries. The Health care sector(in orange) and the Consumer Service Sector(in gray) seems the one mostly connected within the industry, while the finance sector (in purple) is mostly spread in the network. We remark the fact that the absence of the edge does not impl that firms are not correlated, bt it implies that after addjusting for the market effect its partial correlation with other nodes is zero.
